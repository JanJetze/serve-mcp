# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/).

## [0.1.0] - 2026-03-03

### Added

- `serve_file` tool: serve any file over LAN with auto-generated QR code page
- `stop_server` tool: manually stop a running file server
- Auto-close after first download (configurable)
- SVG QR code generation for mobile scanning
- LAN IP auto-detection
- Responsive HTML download page
